# MosStroyInfo — Сервис для заказа и строительства объектов ИЖС

Сервис для полного цикла заказа и строительства объектов индивидуального жилищного строительства (ИЖС). Разработан для партнёра «Мосстройинформ» в рамках хакатона.

## Описание проекта

Сервис обеспечивает полный путь заказчика — от выбора проекта до завершения строительства и подписания финальных документов. Включает в себя:

- **Выбор проекта** — выбор проекта и отправка запроса на строительство
- **Согласование документации** — прохождение этапа согласования документов через систему
- **Этап строительства** — просмотр видеопотока со стройки и общение в чате со специалистом
- **Завершение строительства** — подписание всех оставшихся документов после окончания строительства


### Запуск через Docker Compose

```bash
docker-compose up --build
```
